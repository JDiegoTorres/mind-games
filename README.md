# 🧠 Mind Games

Una colección de juegos mentales construida con HTML, CSS y JavaScript vanilla.

## Juegos disponibles

- 🃏 **Memory** - Encuentra las parejas de cartas
- 🎵 **Simon Says** - Repite la secuencia de colores
- 🧩 **Crucigrama** - Completa las palabras cruzadas con pistas

## Cómo usar

1. Abre `index.html` en tu navegador
2. Elige un juego
3. ¡A jugar!

## Estructura del proyecto

```
mind-games/
├── index.html          ← Página principal
├── css/styles.css      ← Estilos generales
├── js/main.js          ← Lógica general
├── games/memory/       ← Juego Memory
└── README.md           ← Este archivo
```

## Tecnologías

- HTML5
- CSS3
- JavaScript (ES6+)
